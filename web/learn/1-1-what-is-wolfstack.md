# What WolfStack is (and the one thing to remember)

Welcome. If WolfStack felt like a lot the first time you opened it — that's normal, and it's not your fault. There are a lot of buttons on this screen. The good news is you only ever need a few of them at once.

Let's start with what this thing actually *is*.

## The plain-English version

WolfStack is a **control panel for your servers**. It sits on top of the Linux machine you installed it on and gives you buttons for the things you'd otherwise type long commands to do:

- start and stop apps
- create containers and virtual machines
- take backups
- watch for problems
- open a terminal

That's it. It doesn't replace your server. It doesn't lock you in. It's a friendlier front door to a machine you already own.

> If you've used Proxmox: WolfStack plays in the same space, but it's built to let you do the *common* things without learning the whole system first. You can grow into the rest later — or never. Both are fine.

## The one thing to remember

**You do not have to use most of what you see.**

WolfStack is a big toolbox. A big toolbox isn't "overwhelming" — it just has a lot of drawers. You don't empty every drawer to hang a picture. You pick up the one tool you need and ignore the rest.

In this course we'll only ever open one drawer at a time. By the end you'll have:

- a server you understand
- an app running on it
- a backup of it
- an alert if it breaks

…and you'll have ignored about 90% of the screen the whole way through. On purpose.
