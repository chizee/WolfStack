# The map of everything else

You've now done the six core things: found your server, installed an app, built containers, opened a terminal, taken backups, and set up alerts. **You're past the hard part.** Everything below is *optional power* — doors you can open later, when you have an actual reason. Here's a quick map so you know what's behind each one. You don't need to learn any of it today.

Most of these live in the **Apps & Tools** drawer (the grid icon at the top of the sidebar) or under each server.

## The doors

- **VMs** — full virtual machines (a whole separate computer, even Windows or a firewall). Open this when a container isn't enough.
- **WolfNet** — a private, encrypted mesh network that links your servers (and even containers) together as if they were on the same LAN, wherever they are. Great when you have machines in more than one place.
- **Storage / Shares / Storage Array** — mount remote storage (S3, NFS, SMB), share your storage to the rest of your LAN, or build parity disk arrays. For when you outgrow a single disk.
- **App Store (the full catalogue)** — there are **hundreds** of apps in there. You've used it once; it has a lot more.
- **WolfFlow** — visual automation. Drag-and-drop "when this, do that" flows — scheduled backups of *behaviour*, cleanups, update checks, across your whole cluster.
- **WolfAgents** — named AI assistants with memory that can help you operate and answer questions about your setup.
- **Databases** — browse and query MySQL/MariaDB connections without a separate tool.
- **Fleet Security** — fleet-wide security in one screen: password rotation, blocked IPs, exposed ports.
- **Status Pages / Tenants / Cluster Browser** — public uptime pages, multi-customer dashboards, and reaching cluster web apps without a VPN.

## How to explore safely

> **The golden rule:** open a new door only when you have a real, specific need — "I want my two houses' servers to talk" → WolfNet. "I want to automate cleanup" → WolfFlow. Don't tour them all at once. That's the exact overwhelm we've spent this whole course avoiding.

Each of these is learnable the same way you learned the basics: one screen, one task, at a time. You already have the method — you've been using it.

## ✓ What you just learned

- The advanced features (VMs, WolfNet, Storage, WolfFlow, WolfAgents, Databases, Fleet Security…) are **optional power**, mostly in the **Apps & Tools** drawer.
- You don't need any of them to run a healthy setup — you've already learned what you need.
- Open new doors **one at a time, driven by a real need**, not curiosity-overload.
